# Create your own project

If you would like to create a new project you can use the below [github repository](https://github.com/django-superapp/django-superapp-default-project) as a template.

<Steps>
    ### Bootstrap a Project from the Template
    Please setup the project by following the instructions from the [Getting Started](/) guide.

    ### Upload the Project to Github
    Commit your app to a new Github Repository and push it to Github.

    ### Use your App inside your project
    ```shell
    django_superapp bootstrap-project \
        --template-repo https://github.com/<github-username>/<github_repo> \
        ./my_superapp;
    cd my_superapp;
    ```
</Steps>
