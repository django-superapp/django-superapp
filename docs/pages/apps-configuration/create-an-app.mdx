# Create an App

<Steps>
    ### Install django_superapp

    To create a SuperApp app you need `django_superapp` installed. You can do this
    using `pipx`:

    ```shell
    pipx install django_superapp
    ```

    ### Generate the app

    Next, create a new SuperApp app using the `add-app` command

    ```shell
    cd ./my_superapp
    django_superapp add-app --app-name sample_app
    mv superapp/apps/sample_app superapp/apps/your_app_name_here
    ```
</Steps>
