# Bootstrap a new app

If you would like to create a new app you can use the below [github repository](https://github.com/django-superapp/django-superapp-sample-app) as a template.

```shell
cd ./my_superapp;
cd superapp/apps;
django_superapp bootstrap-app --template-repo https://github.com/django-superapp/django-superapp-sample-app ./sample_app;
```
