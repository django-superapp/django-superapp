# Create your own app

If you would like to create a new app you can use the below [github repository](https://github.com/django-superapp/django-superapp-sample-app) as a template.

<Steps>
    ### Setup the project
    Please setup the project by following the instructions from the [Getting Started](/) guide.

    ### Bootstrap an App from the Template
    ```shell
    cd ./my_superapp;
    cd superapp/apps;
    django_superapp bootstrap-app \
        --template-repo https://github.com/django-superapp/django-superapp-sample-app \
        ./sample_app;
    cd ../../;
    ```

    ### Upload the App to Github
    Commit your app to a new Github Repository and push it to Github.

    ### Use your App inside your project
    ```shell
    cd ./my_superapp;
    cd superapp/apps;
    django_superapp bootstrap-app \
        --template-repo https://github.com/<github-username>/<github_repo>\
        ./<app_name>;
    cd ../../;
    ```
</Steps>
