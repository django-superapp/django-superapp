# Admin Portal

This module setup an admin interface based on [django-unfold](https://github.com/unfoldadmin/django-unfold) module.

<Image src="/assets/docs/admin-portal/admin-portal.svg" alt="Admin Portal Screenshot" width={1024} height={1024} className={'mt-3'} />

## Getting Started
<Steps>
### Setup the project
    Please setup the project by following the instructions from the [Getting Started](/) guide.

### Bootstrap the Admin Portal App
```bash
cd superapp/apps;
django_superapp bootstrap-app \
--template-repo https://github.com/django-superapp/django-superapp-admin-portal ./admin_portal;
cd ../../;
```
</Steps>
